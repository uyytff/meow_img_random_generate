# 喵图随机助手 v1.0

> 喵～发送 `/img` 即可从 Lolicon API 获取随机setu喵～

## 功能
- 一行命令 `/img` 获取图片，无需任何参数。
- 管理员可无视冷却，普通用户受冷却限制（默认 300 秒）。
- 支持模糊搜索、图片反代、重试机制等。
- 所有搜索参数均由配置文件统一管理，普通用户无法修改。

## 安装
1. 将插件文件夹放入 AstrBot 的 `data/plugins/` 目录。
2. 重启 AstrBot。
3. 在管理面板中配置插件参数（可实时生效，除 API 地址修改后需重载插件）。

## 配置说明
所有配置项均可在 AstrBot 插件管理界面修改。

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| API | Lolicon API 地址 | https://api.lolicon.app/setu/v2 |
| Proxy 服务地址 | 图片反代地址，解决防盗链 | i.pixiv.re |
| 连续请求 CD | 普通用户两次请求间隔（秒） | 300 |
| API 请求超时 | 单个请求超时时间（秒） | 15 |
| 请求失败重试次数 | 失败后自动重试次数 | 3 |
| r18 | 0=全年龄，1=仅 R18，2=混合 | 0 |
| tag | 图片标签，用 `\|` 分隔 | 空 |
| 排除 AI 作品 | 1=排除，0=包含 | 1 |
| 单次请求数量 | 每次获取图片数量 | 1 |
| 输出大小 | 图片规格（original/regular/small/thumb/mini） | original |
| 模糊搜索 | 0=关闭，1=开启（将单个标签转为关键词搜索） | 0 |
| 缩写自动转换 | 0=禁用，1=允许（API 内置缩写转换） | 1 |

## 使用方法
- 在聊天窗口发送 `/img` 即可。
- 管理员不受冷却限制。
- 如果提示未找到图片，请检查标签设置（如是否过于严格）或排除 AI 设置。

## 注意事项
- 本插件依赖 Lolicon API，请遵守其使用条款。
- Discord 平台建议单次请求数量不超过 10，否则可能被限制。
